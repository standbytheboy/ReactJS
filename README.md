# Pet Shop

Este é o repositório do site do **Pet Shop**, criado para aulas de ReactJS no Senac Lapa Tito. Faremos uma plataforma moderna e interativa construída com **React**.

## Sobre o Projeto

O Pet Shop é um site de demonstração que oferece serviços de banho e tosa, vacinação e venda de produtos para animais de estimação. O objetivo principal deste projeto é apresentar uma interface de usuário responsiva e dinâmica, garantindo uma experiência agradável tanto para os clientes quanto para os tutores dos pets.

## Tecnologias Utilizadas

  * **React:** Biblioteca JavaScript para construir a interface do usuário.
  * **Vite:** Ferramenta de build rápida e moderna para desenvolvimento web.
  * **HTML5 & CSS3:** Linguagens essenciais para a estrutura e estilo do site.
  * **JavaScript:** Linguagem de programação para a lógica da aplicação.

## Funcionalidades

  * **Página Inicial:** Uma visão geral dos serviços e produtos do pet shop.
  * **Serviços:** Detalhes sobre banho, tosa, vacinação e consultas veterinárias.
  * **Produtos:** Uma lista de produtos disponíveis para compra.
  * **Contato:** Formulário para agendamento de serviços ou dúvidas.

## Como Executar o Projeto Localmente

Siga os passos abaixo para ter o projeto rodando na sua máquina.

### Pré-requisitos

Certifique-se de que você tem o **Node.js** e o **npm** (ou **yarn**) instalados.

  * [**Instalar Node.js**](https://nodejs.org/en/download/)

### Instalação

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/seu-repositorio.git
    ```
2.  **Entre no diretório do projeto:**
    ```bash
    cd seu-repositorio
    ```
3.  **Instale as dependências:**
    ```bash
    npm install
    # ou
    yarn install
    ```
4.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    # ou
    yarn dev
    ```

O projeto estará acessível em `http://localhost:5173` (ou em outra porta, conforme indicado no terminal).
